# Installation

## Stable release

To install BestManRobotics, run this command in your terminal:

```sh
uv add bestman
```

Or if you prefer to use `pip`:

```sh
pip install bestman
```

## From source

The source files for BestManRobotics can be downloaded from the [Github repo](https://github.com/write-for-bug/bestman).

You can either clone the public repository:

```sh
git clone git://github.com/write-for-bug/bestman
```

Or download the [tarball](https://github.com/write-for-bug/bestman/tarball/master):

```sh
curl -OJL https://github.com/write-for-bug/bestman/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd bestman
uv pip install .
```
